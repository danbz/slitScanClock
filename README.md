#SlitscanClock

code in C++ using openFrameworks.cc for timeLapse 24hr slitscan clock
included is a compiled binary for OSX

#Controls

* 1: scan horizontal
* 2: scan vertical
* 3: ribbon horizontal
* 4: ribbon vertical
* 5: slitscan clock 
* c: hide preview
* f: toggle fullscreen
* r: draw anystyle scan as radial
* d: draw slitscan image as 3d dots z-ordered by lightness
